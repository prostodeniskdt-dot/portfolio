# Фотографии команды

Положите сюда фотографии участников команды в формате **512×512 пикселей**.

## Требования к изображениям

- **Размер:** 512×512 px
- **Формат:** JPG или PNG
- **Качество:** Высокое (для retina-дисплеев)

## Необходимые файлы

| Файл | Участник | Должность |
|------|----------|-----------|
| `vitaly.jpg` | Виталий Аршук | Основатель BAR BOSS |
| `denis.jpg` | Денис Колодешников | Руководитель ИИ направления |
| `valeria.jpg` | Валерия Колодешникова | Руководитель маркетингового отдела |

## Рекомендации

1. Используйте квадратные фотографии с лицом в центре
2. Фон желательно нейтральный или профессиональный
3. Хорошее освещение лица
4. Формат JPG предпочтительнее для фотографий (меньший размер файла)
