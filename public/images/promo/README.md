# Промо-изображения для пакетов документов

## Куда закидывать фото:

Поместите промо-изображения в эту папку (`public/images/promo/`):

1. **Пакет документов №1**: `package-1-promo.png` (или `.jpg`)
2. **Пакет документов №2**: `package-2-promo.png` (или `.jpg`)

## Требования к изображениям:

- Формат: PNG или JPG
- Рекомендуемый размер: 120x80 пикселей или пропорционально
- Изображения будут отображаться справа от названия продукта в заголовке окна

## Примеры названий файлов:

- `package-1-promo.png`
- `package-1-promo.jpg`
- `package-2-promo.png`
- `package-2-promo.jpg`
