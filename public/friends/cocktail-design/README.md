# Изображения для Cocktail Design

## Структура файлов

Поместите следующие файлы в эту папку:

### Логотип команды
- **Файл:** `logo.png`
- **Размер:** 1280×1280 пикселей
- **Формат:** PNG (рекомендуется) или JPG
- **Описание:** Логотип команды Cocktail Design, который будет отображаться как иконка папки

### Файл "описание"
- **Примечание:** Файл "описание" отображается как документ Word (иконка документа) и не требует загрузки изображения
- **Функция:** При клике на файл "описание" открывается текстовое окно с информацией о компании

### Изображения
- **Файлы:** `1.png`, `2.png`, `3.png`, `4.png`, `5.png`, `6.png`, `7.png`, `8.png`, `9.png`
- **Размер:** 1280×1280 пикселей (или пропорционально)
- **Формат:** PNG (рекомендуется) или JPG
- **Описание:** Изображения, связанные с Cocktail Design

### Видео
- **Файлы:** `video1.mp4`, `video2.mp4`, и т.д.
- **Размер:** 1080×1920 пикселей (вертикальное видео, 9:16)
- **Формат:** MP4 (рекомендуется) или WebM
- **Описание:** Видеофайлы, связанные с Cocktail Design
- **Рекомендации:**
  - Используйте MP4 с кодеком H.264 для лучшей совместимости
  - Сжимайте видео для веба (рекомендуется до 10-20 МБ на файл)
  - Вертикальный формат 1080×1920 идеален для мобильных устройств

## Рекомендации

1. Используйте квадратные изображения (1:1) для лучшего отображения
2. Оптимизируйте изображения для веба (сжатие без потери качества)
3. Для логотипа используйте прозрачный фон (PNG), если это возможно
4. Все изображения будут автоматически адаптированы под мобильные устройства
5. Видео должно быть в вертикальном формате 1080×1920 пикселей (9:16) для оптимального отображения
6. Сжимайте видео для веба (рекомендуется до 10-20 МБ на файл)

## Пример структуры

```
public/
  friends/
    cocktail-design/
      logo.png (1280×1280) - Логотип команды
      1.png (1280×1280) - Изображение 1
      2.png (1280×1280) - Изображение 2
      3.png (1280×1280) - Изображение 3
      4.png (1280×1280) - Изображение 4
      5.png (1280×1280) - Изображение 5
      6.png (1280×1280) - Изображение 6
      7.png (1280×1280) - Изображение 7
      8.png (1280×1280) - Изображение 8
      9.png (1280×1280) - Изображение 9
      video1.mp4 (1080×1920) - Видео 1 (вертикальное)
      video2.mp4 (1080×1920) - Видео 2 (вертикальное)
```

**Примечания:**
- Файл "описание" не требует загрузки изображения - он отображается как иконка документа Word и при клике открывает текстовое окно с информацией о компании
- Видео должно быть в вертикальном формате **1080×1920 пикселей (9:16)** для оптимального отображения на мобильных устройствах
- Для добавления видео в файловый менеджер нужно обновить файл `lib/data/friends.ts`

## Как добавить видео

После загрузки видеофайла (например, `video1.mp4` в формате 1080×1920), добавьте его в `lib/data/friends.ts`:

```typescript
{
  id: "video-cocktail-design-1",
  name: "video1.mp4",
  type: "video",
  filePath: "/friends/cocktail-design/video1.mp4",
  friendId: "cocktail-design",
  order: 10, // Порядок после изображений
},
```
