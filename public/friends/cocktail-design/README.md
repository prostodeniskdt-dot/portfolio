# Изображения для Cocktail Design

## Структура файлов

Поместите следующие файлы в эту папку:

### Логотип команды
- **Файл:** `logo.png`
- **Размер:** 1280×1280 пикселей
- **Формат:** PNG (рекомендуется) или JPG
- **Описание:** Логотип команды Cocktail Design, который будет отображаться как иконка папки

### Файл "описание"
- **Примечание:** Файл "описание" отображается как документ Word (иконка документа) и не требует загрузки изображения
- **Функция:** При клике на файл "описание" открывается текстовое окно с информацией о компании

### Изображения
- **Файлы:** `1.png`, `2.png`, `3.png`, `4.png`, `5.png`, `6.png`
- **Размер:** 1280×1280 пикселей (или пропорционально)
- **Формат:** PNG (рекомендуется) или JPG
- **Описание:** Изображения, связанные с Cocktail Design

## Рекомендации

1. Используйте квадратные изображения (1:1) для лучшего отображения
2. Оптимизируйте изображения для веба (сжатие без потери качества)
3. Для логотипа используйте прозрачный фон (PNG), если это возможно
4. Все изображения будут автоматически адаптированы под мобильные устройства

## Пример структуры

```
public/
  friends/
    cocktail-design/
      logo.png (1280×1280) - Логотип команды
      1.png (1280×1280) - Изображение 1
      2.png (1280×1280) - Изображение 2
      3.png (1280×1280) - Изображение 3
      4.png (1280×1280) - Изображение 4
      5.png (1280×1280) - Изображение 5
      6.png (1280×1280) - Изображение 6
```

**Примечание:** Файл "описание" не требует загрузки изображения - он отображается как иконка документа Word и при клике открывает текстовое окно с информацией о компании.
